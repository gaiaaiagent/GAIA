# .claude Directory

This directory contains Claude's working materials for the RegenAI project - a living workspace where documentation and development are inseparable.

## Structure

- **`journal/`** - Development journal chronicling project progress and insights
  - `00-index.md` - Journal overview and entry summaries
  - `00-journal-template.md` - Template for new entries
  - Numbered entries documenting sessions

- **`planning/`** - Project planning and development frameworks
  - `roadmaps/` - Development milestones and timelines
  - `features/` - Feature specifications and character development
  - `architecture/` - Technical decisions and diagrams (including living documentation workflow)
  - `sprints/` - Sprint planning and tracking
  - `priorities/` - Current and upcoming priorities

## Getting Started

1. **For cold starts**: Begin with the journal index (`journal/00-index.md`) to understand recent work
2. **For planning context**: Check `planning/roadmaps/milestone-1-core-agent-framework.md` for current objectives
3. **For character development**: See `planning/features/character-development-framework.md` for our approach
4. **For workflow understanding**: Read `planning/architecture/living-documentation-workflow.md`

## Living Documentation Principles

This is not a static archive but a living workspace where:
- Documentation happens DURING work, not after
- Questions and uncertainties are documented openly
- Multiple versions show evolution of thinking
- Cross-references create a web of understanding

See CLAUDE.md in the project root for overall project configuration and guidelines.
