Reference Implementations

GitHub Markdown

  • Uses blank lines between paragraphs and lists
  • Lists automatically get proper spacing
  • No blank lines needed between list items
  • Bold/strong works inline in headings and lists

CommonMark Spec

  • Blank line required to separate blocks
  • Lists are block-level elements
  • List items can contain multiple paragraphs
  • Tight vs loose lists determined by blank lines

Claude.ai

  • Uses markdown-it or similar parser
  • Handles lists with and without spacing
  • Good inline formatting support
  • Consistent paragraph spacing
